# 06_ARCHIVE Guide

This folder contains completed projects, old contexts, and historical records.

## What Goes Here

- Completed projects (move entire folder here)
- Old contexts no longer relevant
- Historical documents for reference
- Past experiments

## When to Archive

Move things here when:
- A project is completed or abandoned
- Context is no longer relevant
- You want to preserve but don't need active access

## Naming Convention

When archiving, prefix with date:
- `YYYY-MM-Project-Name/`

## Tags

- #archived - All archived content
- Original tags preserved for searchability
