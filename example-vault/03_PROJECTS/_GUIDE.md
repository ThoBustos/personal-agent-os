# 03_PROJECTS Guide

This folder contains active and past projects with full context.

## Structure

```
03_PROJECTS/
├── _GUIDE.md
└── Project-Name/
    ├── _STATE.md      # Current status, focus, blockers
    ├── _BACKLOG.md    # Tasks, ideas, priorities
    ├── Calls/         # Meeting notes, transcripts
    │   └── (or User/, Client/, Team/ for CRM-style projects)
    ├── Decisions/     # Key decisions with context
    ├── Notes/         # Strategy docs, research
    ├── People/        # Optional: project-specific contacts (CRM)
    ├── Accounts/      # Optional: company/org accounts (CRM)
    └── Legal/         # Optional: contracts, templates (business)
```

## Core Project Files

### _STATE.md
Current state of the project:
- What is the current focus?
- Who is on the team?
- What are the blockers?
- What's next?

### _BACKLOG.md
Ideas, tasks, and priorities:
- Prioritized task list
- Ideas to explore
- Future features

### Calls/
Meeting notes: `YYYY-MM-DD-topic.md`

For CRM-style projects, organize by type:
- `Calls/User/` - End user calls
- `Calls/Client/` - Client/account calls
- `Calls/Team/` - Internal team calls

### Decisions/
Key decisions with context

### Notes/
Strategy docs, research, thinking

## Optional Folders (Business/CRM Projects)

### People/
Project-specific contacts with CRM context:
- Email thread tracking
- Communication preferences
- Company affiliation

### Accounts/
Company/organization tracking:
- Contact list
- Relationship history
- Contract status

### Legal/
Contract workspace:
- `_SYSTEM.md` - Contracting principles
- `Templates/` - Reusable templates
- `Contracts/` - Signed agreements
- `Strategies/` - Legal strategy docs
- `Clients/` - Client-specific legal context
