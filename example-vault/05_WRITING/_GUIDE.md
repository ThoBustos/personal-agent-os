# 05_WRITING Guide

This folder is for your writing - reflections, drafts, published work, and ideas.

## Structure

```
05_WRITING/
├── Reflections/    # Personal thinking
├── Drafts/         # Work in progress
├── Published/      # Shipped work
└── Ideas/          # Seeds and hooks
```

## Subfolders

### Reflections/
Personal writing not meant for publication.

### Drafts/
Work in progress content.

### Published/
Archive of shipped work.

### Ideas/
Seeds for future content - one-liners, hooks, questions.

## Philosophy

> **Writing > Reading** - Writing is processing. It forces clarity.

> **Creation > Consumption** - Create more than you consume.
