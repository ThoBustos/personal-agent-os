# 00_SYSTEM Guide

This folder contains the core operating system for your life. It defines who you are, what matters, and how you operate.

## Files

| File | Purpose | Update Frequency |
|------|---------|------------------|
| `GLOBAL_STATE.md` | Current focus, energy, active projects | Weekly or as needed |
| `GLOBAL_STATE_ARCHIVE.md` | Historical states for reflection | When archiving |
| `LIFE_VISION.md` | End-game identity, north star | Yearly |
| `LIFE_VISION_ARCHIVE.md` | Past visions for evolution tracking | When vision changes |
| `PILLARS.md` | The 10 life pillars defined | Rarely |
| `PRINCIPLES.md` | Operating principles and frameworks | Rarely |

## OPS/ Subfolder

Contains personal processes and rituals:
- `daily-routine.md` - Morning/evening routines
- `weekly-ritual.md` - Weekly review process

## For Claude

When helping the user:
1. Read `GLOBAL_STATE.md` first to understand current focus
2. Reference `PILLARS.md` when discussing life balance
3. Use `PRINCIPLES.md` to align recommendations with values
4. Check `LIFE_VISION.md` for long-term direction

## Tags Used Here

- #active - Currently active focus areas
- #pillar/* - References to life pillars
