# 04_PEOPLE Guide

This folder contains notes about people in your life.

## Structure

One note per person: `Name.md`

## Note Template

```markdown
# {{Name}}

## Who They Are
- Met: {{date}}
- Context: {{how you know them}}
- Personal: {{interests, location, family}}

## Projects
- [[Project]] - {{role}}

## Interactions
- YYYY-MM-DD: {{topic}}

## Tags
#person #project/{{name}}
```

## What to Track

- How you met and context
- Role in projects
- Key interactions
- Working style
- Relationship maintenance

## Tags

- #person - All people notes
- #project/* - Project associations
- #mentor, #colleague, #friend, #family
