# 02_JOURNAL Guide

This folder contains reflection documents at different cadences.

## Structure

```
02_JOURNAL/
├── Weekly/
│   └── YYYY-W##.md    # Weekly docs contain daily entries
└── Monthly/
    └── YYYY-MM.md     # Monthly calibration
```

## Weekly Documents

Each weekly doc contains:
1. **Week intentions** (keystone focus, pillar coverage, brave act, joy anchor, system upgrade)
2. **Daily sections** (Mon-Sun) for quick entries
3. **Weekly reflection** at the end

## Monthly Documents

Monthly docs answer the 8 reflection questions:
1. Identity Calibration
2. Energy Audit
3. Truth Inventory
4. Relationship Scan
5. Creation Review
6. Leverage Check
7. Fear & Edge Detection
8. Wonder Scan

## Naming Convention

- Weekly: `YYYY-W##.md` (ISO week numbers)
- Monthly: `YYYY-MM.md`

## Tags Used Here

- #gratitude - Gratitude entries
- #memory - Significant moments
- #pattern/* - Emergent patterns
